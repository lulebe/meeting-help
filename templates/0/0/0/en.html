Page doesn't load correctly

First, try reloading the page.
If that doesn't fix the problem, make sure the minimum requirements for heiconf are all met:

  • Browser: Use an up-to-date version of Mozilla Firefox, Google Chrome or Microsoft Edge (Version 80+), preferable on a Desktop OS (Windows, macOS, Linux, Chrome OS).
  • Connection: Up 1 MBit/s, Down 2 MBit/s (you can check your connection on speedtest.net
If your connection speed is too slow, consider using a wired connection or move closer to your Wi-Fi access point.
