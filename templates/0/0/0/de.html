Seite lädt nicht korrekt

Prüfen Sie zunächst, ob das Problem nach erneutem Laden der Seite fortbesteht.
Wenn ja, dann überprüfen Sie die Mindestanforderungen für heiconf:

  • Browser: Eine aktuelle Version von Mozilla Firefox, Google Chrome oder Microsoft Edge (Version 80+), am Besten auf einem Desktop Betriebssystem (Windows, macOS, Linux, Chrome OS), ist empfohlen.
  • Verbindung: Up 1 MBit/s, Down 2 MBit/s (kann auf speedtest.net getestet werden).
Im Falle einer zu langsamen Verbindung, probieren Sie eine kabelgebundene Verbindung aus oder versuchen Sie, den WLAN- bzw. LTE-Empfang zu verbessern.
