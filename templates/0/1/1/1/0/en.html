Windows 10 Audio Setup

Open the Windows settings: Navigate to the System settings:

Open the "audio" tab: Check the following settings:

  1. The correct output should be set (your speakers / headphones)
  2. The volume should be high enough
  3. The correct input has to be set (your microphone / headset)
  4. You can run a microphone test here.

After adjusting these settings, you should rejoin the meeting by reloading the page.
