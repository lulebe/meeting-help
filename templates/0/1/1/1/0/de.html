Windows 10 Audio Setup

Öffnen Sie zunächst die Windows Einstellungen: Und navigieren Sie zu den System-Einstellungen:

Dort gibt es im Reiter der Audio-Einstellungen einige Optionen: Achten Sie auf Folgendes:

  1. Das korrekte Ausgabegerät muss gewählt sein (Ihr Kopfhörer / Lautsprecher)
  2. Die Lautstärke muss laut genug sein
  3. Der korrekte Eingang muss gewählt sein (Ihr Mikrofon / Headset)
  4. Ein Mikrofontest kann bei Bedarf direkt hier durchgeführt werden.

Nach dem Anpassen der Einstellungen sollten Sie dem Raum durch Neuladen der Seite erneut beitreten.
