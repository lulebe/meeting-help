Ich höre andere nicht

Stellen Sie zunächst sicher, dass Sie der Audiokonferenz beigetreten sind. Falls Sie dennoch nichts hören, verlassen Sie sie und treten Sie erneut bei, indem Sie den folgenden Button nutzen (das Symbol unterscheidet sich je nach Status: nicht beigetreten, nur zuhören, mit Mikrofon):

Danach sollten Sie probieren, dem Raum durch Neuladen der Seite erneut beizutreten.
Falls das nicht hilft, überprüfen Sie folgende Einstellungen:

  • Audio Einstellungen: Stellen Sie sicher, dass ihr Gerät auf den richtigen Audio-Ausgang eingestellt ist und die Lautstärke nicht auf "stumm" steht.
  • Browser Einstellungen: Manche Browser erlauben es, einzelne Tabs stummzuschalten. Dann sollte ein Rechtsklick auf den Tab in der Tableiste (ganz oben im Browser-Fenster) eine Option zum anschalten der Tonausgabe in diesem Tab bieten.
