macOS Audio Setup

Öffnen Sie die macOS Systemeinstellungen und navigieren Sie zu den Toneinstellungen:

Dort können Sie für die Ausgabe folgendes einstellen: Achten Sie auf Folgendes:

  1. Das korrekte Ausgabegerät muss gewählt sein (Ihr Kopfhörer / Lautsprecher)
  2. Die Lautstärke muss laut genug sein

Im Reiter der Eingabe stellen Sie Folgendes ein: Achten Sie auf Folgendes:

  1. Das korrekte Eingabegerät muss gewählt sein (Ihr Mikrofon / Headset)
  2. Die Lautstärke muss laut genug sein
  3. Unterhalb des Lautsträkeregler gibt es eine grafische Anzeige des Eingangs, mit dem Sie erkennen können, ob das Mikrofon funktioniert.

Nach dem Anpassen der Einstellungen sollten Sie dem Raum durch Neuladen der Seite erneut beitreten.
