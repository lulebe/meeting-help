macOS Audio Setup

Open the macOS system preferences and navigate to the sound settings:

Open the "Output" tab: Check the following settings:

  1. The correct output should be set (your speakers / headphones)
  2. The volume should be high enough

Open the "Input" tab: Check the following settings:

  1. The correct output should be set (your microphone / headset)
  2. The volume should be high enough
  3. Underneath the volume slider you can visually check whether your microphone is working correctly.

After adjusting these settings, you should rejoin the meeting by reloading the page.
