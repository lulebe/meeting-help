I can't hear other participants

First, try re-joining the room by reloading the page. If that doesn't help, check the following settings:

  • Audio settings: Make sure your device is set to the right audio output and the volume is not muted.
  • Browser settings: Some browser allow muting a tab. Right-click the heiconf browser tab in the tab bar (top of your browser window) and check if you can un-mute it there.
If the problem persists, click below to get further help.
